# Symbols Starter Kit
Example dev setup (boilerplate) to use [DOMQL](https://github.com/domql/domql). You can also check live editor [Playground](https://domql.com/playground/).

### Setup

1. Clone the repo
```
git clone git@github.com:rdourado/symbols.git
```

2. Install scripts
```
npm install
```

3. Run the project
```
npm start
```
